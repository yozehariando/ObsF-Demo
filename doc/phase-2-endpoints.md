# Phase 2 Endpoints

## Sequence Embedding

### Description

Upload and embed a DNA sequence file using machine learning models.

This endpoint processes a DNA sequence file and generates embeddings using the specified model. The process is asynchronous - a job ID is returned that can be used to poll for completion.

Args: file (UploadFile): FASTA or plain text file containing the DNA sequence embedding_model (str, optional): Model to use for embedding. Defaults to DNABERT-S. reduced (bool, optional): Whether to use reduced dimensionality embeddings. Defaults to True.

Returns: EmbeddingResponse: Contains job_id, sequence_hash, and status information

Example:

```code
files = {'file': open('sequence.fasta', 'rb')}
data = {'embedding_model': 'DNABERT-S', 'reduced': True}
response = requests.post('/api/v1/pathtrack/sequence/embed', files=files, data=data)
job_id = response.json()['job_id']
```

Raises: HTTPException: 400 if sequence is invalid, 500 for processing errors

### Endpoint

```text
curl -X 'POST' \
  'http://54.169.186.71/api/v1/pathtrack/sequence/embed' \
  -H 'accept: application/json' \
  -H 'X-API-Key: test_key' \
  -H 'Content-Type: multipart/form-data' \
  -F 'file=@rpoB.fa' \
  -F 'embedding_model=DNABERT-S' \
  -F 'reduced=true'
```

### Response

```json
{
    "job_id": "551e540ed5074aa788e2f857df371b71",
    "sequence_hash": "a45d3c548bf1dc33c5440936f155b36c",
    "embedding_model": "DNABERT-S",
    "s3_path": "s3://pathgen-public-genomic-dev/embeddings/DNABERT-S/a45d3c548bf1dc33c5440936f155b36c_reduced.npy",
    "is_cached": true,
    "is_ready": true
}
```

## Check Job Status

### Description

Check the status of an asynchronous background job.

This endpoint is used to poll for completion of any asynchronous job (embedding, UMAP, etc.). For embedding jobs, it checks S3 for completion of the embedding process.

Args: job_id (str): The ID of the job to check

Returns: Job: Current status of the job including completion state and results

Example: 

```code
response = requests.get(f'/api/v1/pathtrack/jobs/{job_id}')
status = response.json()['status']  # 'PENDING', 'RUNNING', 'COMPLETED', or 'FAILED'
```

Raises: HTTPException: 404 if job not found, 500 for processing errors

### Endpoint

```text
curl -X 'GET' \
  'http://54.169.186.71/api/v1/pathtrack/jobs/f81fc9789a3fedaed55aac2babc8be04' \
  -H 'accept: application/json' \
  -H 'X-API-Key: test_key'
```

```json
{
    "id": "f81fc9789a3fedaed55aac2babc8be04",
```

## Transform Embedding

### Description

Apply UMAP dimensionality reduction to a sequence embedding.

Takes a completed embedding job and generates 2D coordinates using UMAP for visualization. The process is asynchronous and may take time for large embeddings.

Args: job_id (str): ID of a completed embedding job

Returns: Job: Job status with UMAP coordinates in the result when complete

Example:

```code
# First get an embedding job ID
embedding_response = requests.post('/api/v1/pathtrack/sequence/embed', ...)
embedding_job_id = embedding_response.json()['job_id']

# Wait for embedding to complete, then request UMAP
umap_response = requests.post(f'/api/v1/pathtrack/sequence/umap?job_id={embedding_job_id}')
umap_job_id = umap_response.json()['job_id']
```

Raises: HTTPException: 400 for invalid job, 404 if job not found, 500 for processing errors

### Endpoint

```text
curl -X 'POST' \
  'http://54.169.186.71/api/v1/pathtrack/sequence/umap?job_id=f81fc9789a3fedaed55aac2babc8be04' \
  -H 'accept: application/json' \
  -H 'X-API-Key: test_key' \
  -d ''
```

## Find Similar Sequences

### Description

Find sequences similar to a given sequence using vector similarity search.

Uses Vector Similarity Search to find sequences with similar embeddings to the input sequence. Results can be filtered by collection year and similarity threshold.

Args: job_id (str): ID of a completed embedding job similarity_request (SimilarityRequest): Search parameters including: - n_results: Number of similar sequences to return - min_distance: Minimum similarity threshold (-1.0 to 1.0) - max_year: Maximum collection year to include - include_unknown_dates: Whether to include sequences with unknown dates

Returns: Job: Job status with similar sequences in the result when complete

Example: 

```code
params = {
    "n_results": 5,
    "min_distance": 0.8,
    "max_year": 2023,
    "include_unknown_dates": False
}
response = requests.post(
    f'/api/v1/pathtrack/sequence/similar?job_id={job_id}',
    json=params
)
```

Raises: HTTPException: 400 for invalid parameters, 404 if job not found, 500 for processing errors

### Endpoint

```text
curl -X 'POST' \
  'http://54.169.186.71/api/v1/pathtrack/sequence/similar?job_id=f81fc9789a3fedaed55aac2babc8be04' \
  -H 'accept: application/json' \
  -H 'X-API-Key: test_key' \
  -H 'Content-Type: application/json' \
  -d '{
  "n_results": 3,
  "min_distance": -1,
  "max_year": 0,
  "include_unknown_dates": false
}'
```

## Get Embedding From Job Id

### Description

Get the raw embedding vector data from a completed embedding job.

Retrieves the actual embedding vector and metadata from S3 storage. The job must be a completed sequence embedding job.

Args: job_id (str): ID of a completed embedding job

Returns: EmbeddingDataResponse: Contains embedding vector, shape, and metadata

Example:

```text
response = requests.get(f'/api/v1/pathtrack/embedding/{job_id}')
embedding_vector = response.json()['embedding']  # numpy array as list
shape = response.json()['shape']  # original shape of the embedding
```

Raises: HTTPException: 400 for invalid job, 404 if not found, 500 for processing errors

### Endpoint

```text
curl -X 'GET' \
  'http://54.169.186.71/api/v1/pathtrack/embedding/f81fc9789a3fedaed55aac2babc8be04' \
  -H 'accept: application/json' \
  -H 'X-API-Key: test_key'
```
